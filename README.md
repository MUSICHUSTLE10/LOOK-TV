# LOOK TV — Streaming TV & Radio Mondial

Plateforme de streaming IPTV et radio mondiale développée par **LOOK TECHNOLOGIE**.

---

## Description

LOOK TV est une application web de streaming en direct qui permet d'accéder à des milliers de chaînes de télévision et de stations de radio du monde entier, directement depuis un navigateur web, sans installation requise.

---

## Fonctionnalités

- **TV en direct** — Plus de 10 000 chaînes IPTV mondiales en streaming HLS
- **Radio mondiale** — Plus de 5 000 stations de radio internationales
- **Recherche** — Recherche instantanée par nom de chaîne ou pays
- **Filtres** — Filtrage par pays (195 pays) et par catégorie / genre
- **Favoris** — Sauvegarde locale des chaînes et radios favorites
- **Historique** — Accès rapide aux chaînes récemment regardées
- **Qualité automatique** — Sélection automatique de la meilleure résolution disponible
- **Interface adaptative** — Barre latérale réductible pour agrandir l'écran
- **Plein écran** — Mode plein écran pour une expérience cinéma

---

## Technologies utilisées

| Technologie | Usage |
|---|---|
| HTML5 / CSS3 | Structure et mise en page |
| JavaScript (Vanilla) | Logique applicative |
| HLS.js | Lecture des flux vidéo HLS |
| Font Awesome 6 | Icônes |
| Google Fonts (Inter) | Typographie |
| IPTV-org API | Source des chaînes TV |
| Radio Browser API | Source des stations radio |

## Lancement

Ouvrir le fichier `index.html` dans un navigateur web moderne (Chrome, Firefox, Edge).

> Aucune installation ni serveur requis. Une connexion internet est nécessaire pour charger les flux en direct.
